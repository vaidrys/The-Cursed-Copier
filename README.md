# L'Épopée du Copieur Maudit

A minimalist turn-based RPG made in pure Python (Tkinter GUI) where you embody a doomed programmer trapped in an endless cycle of debugging and copying code.

## Core features

 - Village hub with merchant (buy potions & gear)
 - Dark forest exploration with random encounters
 - Turn-based combat against thematic enemies: Bug Errant, Erreur 404, Infinite Loop
 - Stats progression: HP, ATK, DEF, gold, XP → level-ups with permanent boosts
 - Inventory system (potions for healing)
 - Retro dark theme + colored text messages

## Gameplay loop

Buy potions → explore forest → fight bugs → level up → get stronger
